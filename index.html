Conhecimentos

Html

Através do HTML é possivel inserir todo conteudo vísivel na página. É nele que se aplica a técnica de SEO e deixa os sites no topo dos mecanismos de buscas. Com uma estruturação super atualizada e organizada desenvolvo qualquer tipo de layout.

Css

CSS é uma "folha de estilo" composta por “camadas” e utilizada para definir a apresentação (aparência) em páginas da internet. É aqui que aplico todos os estilos e uso um pouco da minha criatividade para deixar o site perfeito para o usuário.

JavaScript

Com a linguagem de programação Java Script é onde relamente dou vida para o site, ela me permite implementar itens complexos na pagina web. É aqui que uso toda minha lógica para o site fazer mais do que simplesmente mostar uma informação para o usuário.

Node.js

A maioria das aplicações hoje em dia necessitam do lado do servidor. Com isso Utilizo muito o Node.js para rodar minhas aplicações em servidores abertos em fazer API. Com o Node.js consigo programar em Java Script que gosto muito, alem de ele ser muito leve é multiplataforma.

React.js

Com o React.js que é uma biblioteca de JavaScript, open source, utilizo para construir user interfaces, nomeadas para aplicações de página única. É aqui que dou aquele toque a mais para deixar o usuario da minha aplicação super entretido oferencendo a ele uma resposta excelente.

Git

O Git é indispensavel para qualquer projeto de desenvolvimento de software. É nele que vem todo o controle de versão de arquivos, alem de facilitar o trabalho em equipe onde podemos editar e criar novos arquivos sem riscos de alterações indesejadas.
